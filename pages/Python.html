Introdução ao Python

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. É amplamente utilizada para desenvolvimento web, automação, análise de dados, inteligência artificial e muito mais.

Funções

Funções em Python permitem organizar o código em blocos reutilizáveis. São definidas com a palavra-chave def.

Loops

Python suporta loops for e while, usados para repetir blocos de código enquanto uma condição é verdadeira.

Exercícios Funções

  1. Faça uma função que leia um vetor de 5 números inteiros e mostre-os.
  2. Faça uma função que leia um vetor de 10 números reais e mostre-os na ordem inversa.
  3. Faça uma função que leia 4 notas, mostre as notas e a média na tela.
  4. Faça uma função que leia um vetor de 10 caracteres, e diga quantas consoantes foram lidas. Imprima as consoantes.
  5. Faça uma função que leia 20 números inteiros e armazene-os num vetor. Armazene os números pares no vetor PAR e os números ímpares no vetor IMPAR. Imprima os três vetores.
  6. Faça uma função que peça as quatro notas de 10 alunos, calcule e armazene num vetor a média de cada aluno, imprima o número de alunos com média maior ou igual a 7.0.
  7. Faça uma função que leia um vetor de 5 números inteiros, mostre a soma, a multiplicação e os números.
  8. Faça uma função que peça a idade e a altura de 5 pessoas, armazene cada informação no seu respectivo vetor. Imprima a idade e a altura na ordem inversa à ordem lida.
  9. Faça uma função que leia um vetor A com 10 números inteiros, calcule e mostre a soma dos quadrados dos elementos do vetor.
  10. Faça uma função que leia dois vetores com 10 elementos cada. Gere um terceiro vetor de 20 elementos, cujos valores deverão ser compostos pelos elementos intercalados dos dois outros vetores.
